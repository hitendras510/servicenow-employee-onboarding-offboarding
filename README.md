# ServiceNow Employee Onboarding & Offboarding Automation

## Overview
This project automates the employee lifecycle using ServiceNow.

The system manages onboarding and offboarding requests by automating approvals and assigning tasks to HR, IT, Facilities, and Security teams using ServiceNow Flow Designer.

## Features
- Employee onboarding request automation
- Employee offboarding workflow
- Approval workflows
- Department task assignments
- Lifecycle tracking

## Workflow

Employee Onboarding
Employee request → Manager approval → HR processing → IT account setup → Equipment allocation → Employee ready

Employee Offboarding
Offboarding request → Manager approval → IT disables access → Facilities collects assets → HR updates records
