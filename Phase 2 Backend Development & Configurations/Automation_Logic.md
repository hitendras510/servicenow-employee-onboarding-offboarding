# Automation Logic

**Project:** Automated Employee Onboarding & Offboarding System
**Phase:** Automation Logic Implementation

---

## Description

Flow Designer automates onboarding and offboarding processes by routing requests for manager approval and generating tasks for HR, IT, Facilities, and Security teams. Status updates, SLA tracking, and email notifications are triggered automatically to ensure timely execution and proper coordination among departments.

---

## Subtasks

### 1. Creation of Flow

**Objective**
Create automated flows to manage onboarding and offboarding processes.

**Tasks**

* Design flows using **ServiceNow Flow Designer**.
* Trigger flows when a new onboarding or offboarding request is submitted.
* Route requests to the **manager for approval**.
* Start workflow execution after approval.

**Expected Result**

* Automated routing of employee lifecycle requests.
* Reduced manual intervention in request processing.

---

### 2. Creation & Implementation of Flows and Actions in Flow Designer

**Objective**
Implement workflow actions that automate task creation and system notifications.

**Tasks**

* Generate tasks for **HR**, **IT**, **Facilities**, and **Security** teams.
* Update request status automatically during each workflow stage.
* Configure **SLA tracking** for task completion.
* Send **email notifications** to stakeholders.
* Ensure all tasks are completed before closing the request.

**Expected Result**

* Automated departmental task assignments.
* Real-time status updates for requests.
* Improved coordination between departments.
* Faster onboarding and offboarding process execution.

---

## Expected Outcome

* Fully automated employee onboarding and offboarding workflows.
* Faster approval and task assignment process.
* Improved transparency through automated notifications and status updates.
* Better SLA compliance and process efficiency.
