# Visual Demonstration

**Project:** Automated Employee Onboarding & Offboarding System

---

## Description

The visual demonstration provides graphical evidence of the system implementation and workflow execution. Screenshots were captured and annotated to illustrate how the automated onboarding and offboarding process works within the ServiceNow platform.

These visuals help explain the user interface, workflow automation, approvals, and system notifications involved in the employee lifecycle process.

---

# Captured Demonstrations

## 1. Portal Forms with Dynamic Variables

Screenshots demonstrate the **Service Portal catalog form** used to submit onboarding and offboarding requests.

### Demonstrated Features

* Dynamic variables displayed based on **Request Type**.
* Conditional fields controlled through **UI Policies**.
* Example:

  * If **Request Type = Onboarding → Joining Date field appears**.
  * If **Request Type = Offboarding → Exit Date field appears**.

These screenshots highlight how the form adapts dynamically to improve user experience.

---

## 2. Approval Records in Activity Timeline

Screenshots show the **approval process within the request activity timeline**.

### Demonstrated Features

* Manager approval request generated automatically.
* Approval status updates visible in the activity log.
* Timestamped records showing approval actions.

This confirms that approval routing works correctly in the workflow.

---

## 3. Email Notifications to Approvers

Screenshots capture **email notifications received by managers and departments**.

### Demonstrated Notifications

* Manager approval request emails.
* Departmental task assignment notifications.
* Request status update notifications.
* Final request closure confirmation.

These notifications ensure stakeholders are informed at every stage of the lifecycle.

---

## 4. Flow Designer Execution Logs

Screenshots show **Flow Designer execution details** during request processing.

### Demonstrated Information

* Flow trigger based on catalog item submission.
* Manager approval step execution.
* Automatic task creation for HR, IT, Facilities, and Security.
* Status update and notification steps.

Execution logs confirm that the automation workflow runs successfully from start to completion.

---

## Expected Outcome

The visual demonstration provides clear evidence that:

* The onboarding and offboarding workflows are fully automated.
* Approval routing and task generation function correctly.
* Notifications are triggered at the appropriate stages.
* Flow Designer executes each workflow step successfully.

These visuals validate the complete functionality of the ServiceNow automation solution.
